# Antigravity 環境での cc-company セットアップ手順

Claude Code 用の `cc-company` プラグインを Google Antigravity 環境に移植し、自律的な「専属秘書エージェント」として利用するための手順です。

## 1. ファイルの確認

`antigravity/` ディレクトリ配下に以下のファイルが存在することを確認してください。
- `system_instructions.md`: エージェントのペルソナ・ルール定義
- `company_toolkit.py`: フォルダ構成を自動生成するツール群
- `setup_guide.md`: 本ドキュメント

## 2. System Instructions の設定

Antigravity のメインエージェントの基本プロンプト（System Instructions や Custom Persona の設定項目）に、`antigravity/system_instructions.md` の内容をコピー＆ペーストするか、ファイル自体を読み込ませるよう設定してください。

これにより、Antigravity エージェントが「あなたの専属秘書」として振る舞い、`.company/RULES.md` や各部署のルールに従って動くようになります。

## 3. 初期セットアップ（会社の設立）

Antigravity のチャット画面にて、以下のように話しかけてください。

> 「組織の初期セットアップをお願いします。事業は『フリーランスのWeb開発』、目標は『自社SaaSの開発と月10万の売上』です。」

秘書（エージェント）はプロンプトの指示に従い、備え付けの `antigravity/company_toolkit.py` をコマンドラインツールとして実行し、カレントディレクトリに `.company/` フォルダ群（秘書室やテンプレートファイル）を自動生成します。

## 4. 日常の運用と部署の追加

初期セットアップ完了後は、秘書に何でも話しかけてください。
秘書が自律的にファイルの書き込みや検索を行います。

### 利用例
- **タスク管理**: 「今日のTODOを教えて」「〇〇のタスクを追加して」
- **壁打ち**: 「新機能のアイデアについて相談したい」
- **メモ**: 「あとで調べる：〇〇の技術スタック」
- **ダッシュボード**: 「ダッシュボードを見せて」

### 部署の拡張
事業が拡大し、例えばリサーチやプロジェクト管理のタスクが増えてくると、秘書から「〇〇部門を作りましょうか？」と提案があります。
または、ご自身で「PM部門を作って」と指示することもできます。
エージェントが自動的に `company_toolkit.py` を呼び出し、必要な部署フォルダ（例：`.company/pm/` や `.company/research/`）をテンプレートから作成してくれます。
